# AGENTS.md

## Project
Litera adalah literature search engine untuk referensi penelitian mahasiswa.

## Important Rules
- Baca PRD.md sebelum melakukan perubahan.
- Pertahankan tampilan visual hasil export Figma Make.
- Jangan mengubah desain secara drastis tanpa alasan teknis.
- Gunakan komponen UI yang sudah ada jika memungkinkan.
- Jangan mengganti seluruh frontend hanya untuk menyederhanakan implementasi.
- Kerjakan fitur secara bertahap.
- Setelah setiap tahap, jalankan lint, type check, build, dan test yang relevan.
- Jangan menggunakan Elasticsearch, embedding, vector database, atau semantic search.
- Implementasikan inverted index dan TF-IDF secara eksplisit agar metode information retrieval dapat dipresentasikan.
- Jangan memasukkan secret ke repository.
- Dokumentasikan cara menjalankan proyek dalam README.md.
